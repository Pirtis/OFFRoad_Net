# OFF-Road Net: Автономное управление вездеходом в приарктических условиях

## 📌 О проекте

Проект посвящен разработке **RL-агента** (Reinforcement Learning) для автономного управления вездеходом в условиях Арктики. Агент обучается объезжать препятствия и достигать цели на основе данных с лидаров, GPS и IMU, без использования заранее построенной карты.

## 🎯 Цель

Создать алгоритм, позволяющий вездеходу самостоятельно ориентироваться на местности, избегать столкновений с препятствиями (камни, деревья, снежные заносы) и двигаться к заданной точке в условиях отсутствия чётких ориентиров.

## 🧠 Как это работает

### Архитектура решения
Сенсоры (лидары, GPS, IMU)  
↓  
Формирование состояния (12 параметров)  
↓  
RL-агент (алгоритм PPO / SAC)  
↓  
Управление (скорость, угол поворота)  
↓  
Движение к цели + объезд препятствий  

### Основные компоненты

| Компонент | Описание |
|-----------|----------|
| **Среда симуляции** | Webots (физически корректная симуляция) |
| **Алгоритмы RL** | PPO и SAC (сравнение эффективности) |
| **Сенсоры** | 3 лидара (фронт, лево, право), GPS, IMU |
| **Наблюдение** | 12 параметров (позиция, углы, лидары, скорость) |
| **Действия** | скорость (-5..20 км/ч) и угол поворота руля (-0.8..0.8) |

## 📊 Система наград (Reward)

| Действие | Награда |
|----------|---------|
| Приближение к цели | +50/метр |
| Удаление от цели | -30/метр |
| Достижение цели | +1000 + бонус |
| Столкновение | -800 |
| Близость к препятствию | до -40 |
| Движение | +0.3 |
| Остановка | -0.5 |
| Штраф за шаг | -0.2 |

## 🚀 Запуск проекта

### 1. Клонирование репозитория
```bash
git clone https://github.com/Pirtis/OFFRoad_Net.git
cd OFFRoad_Net
```
### 2. Создание виртуального окружения
```
python -m venv venv
source venv/bin/activate      # Linux/Mac
venv\Scripts\activate         # Windows
```
### 4. Установка зависимостей
```
pip install -r requirements.txt
```
### 6. Запуск обучения
```
cd PPO/controllers/rl_controler_big
python rl_controler_big.py
```
### 7. Запуск тестирования обученной модели
bash
```
python trained_rl.py --test
```
