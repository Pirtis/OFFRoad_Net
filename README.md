# OFF-Road Net: Автономное управление вездеходом в приарктических условиях

## 📌 О проекте

Проект посвящен разработке **RL-агента** (Reinforcement Learning) для автономного управления вездеходом в условиях Арктики. Агент обучается объезжать препятствия и достигать цели на основе данных с лидаров, GPS и IMU, без использования заранее построенной карты.

## 🎯 Цель

Создать алгоритм, позволяющий вездеходу самостоятельно ориентироваться на местности, избегать столкновений с препятствиями (камни, деревья, снежные заносы) и двигаться к заданной точке в условиях отсутствия чётких ориентиров.

## 🧠 Как это работает

### Архитектура решения
Сенсоры (лидары, GPS, IMU)  
↓  
Формирование состояния (12 параметров)  
↓  
RL-агент (алгоритм PPO / SAC)  
↓  
Управление (скорость, угол поворота)  
↓  
Движение к цели + объезд препятствий  

### Основные компоненты

| Компонент | Описание |
|-----------|----------|
| **Среда симуляции** | Webots (физически корректная симуляция) |
| **Алгоритмы RL** | PPO и SAC (сравнение эффективности) |
| **Сенсоры** | 3 лидара (фронт, лево, право), GPS, IMU |
| **Наблюдение** | 12 параметров (позиция, углы, лидары, скорость) |
| **Действия** | скорость (-5..20 км/ч) и угол поворота руля (-0.8..0.8) |

## 📊 Система наград (Reward)

| Действие | Награда |
|----------|---------|
| награда за приближение к цели | +5/метр |
| награда за прохождение расстояния | +1 |
| большая награда за достижение цели | +50 |
| штраф за пересечение траектории с препятствиями | - (макс. длина луча – длина луча)/1.5 |
| штраф за низкую скорость, стояние на месте | (2 – скорость), если скорость < 1 |
| большой штраф за столкновение | -50 |

├── SAC/                      # Контроллеры с алгоритмом SAC  
│   ├── plugins/robot_windows/ # Плагины Webots  
│   ├── worlds/ # Миры для SAC  
│   ├── algoritm_sac.py # Реализация SAC алгоритма

## 🚀 Запуск проекта

### 1. Клонирование репозитория
```bash
git clone https://github.com/Pirtis/OFFRoad_Net.git
cd OFFRoad_Net
```
### 2. Создание виртуального окружения
```
python -m venv venv
source venv/bin/activate      # Linux/Mac
venv\Scripts\activate         # Windows
```
### 4. Установка зависимостей
```
pip install -r requirements.txt
```
### 6. Запуск обучения
Чтобы запустить обчуение, нужно открыть и запустить соответствующий мир в Webots (расширение файла wbt)
| Мир | Назначение |
|----------|---------|
| training_2(тест обученных моделей).wbt | Для тестирования моделей на простой карте(PPO)  |
| training_3_random(1 obstacle).wbt | Для обучения моделей на карте с 1 рандомногенерируемым препятствием(PPO)   |
| training_3_random(1-4 obstacle).wbt | Для обучения моделей на карте с 1-4 рандомногенерируемыми препятствиями(PPO)  |
| training_3_random(1-4 obstacle)тест модели.wbt | Для тестирования моделей на карте с рандомногенерируемыми препятствиями от 1 до 4(PPO)  |
| training_3_random_big.wbt | Для обучения модли на большой карте с множеством статичных препятствий(PPO)  |
| moose_demo.wbt | Для обучения модли на большой карте с множеством статичных препятствий(SAC)   |
